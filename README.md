# Obsidian.md OSINT Templates

Case report template to use with [Obsidian.md](https://obsidian.md)

## Requirements

- Install and enable the core plugin: `Templates`

```bash
OBSIDIAN_VAULT/
# Add folder templates
mkdir OBSIDIAN_VAULT/templates
wget https://raw.githubusercontent.com/supasuge/OSINT-templates/refs/heads/main/OSINT-case-report.template.md -O OBSIDIAN_VAULT/templates/OSINT-case-report
```

- Make sure to replace `OBSIDIAN_VAULT` with the name of your actual obsidian vault as it is in your local file system. Absolute path is reccomended
